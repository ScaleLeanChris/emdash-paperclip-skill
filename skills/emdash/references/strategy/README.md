# Content Strategy

Plan what to write, when to publish, and how to build an effective editorial cadence. This domain is workflow-driven — it uses emdash content APIs to audit existing content and schedule new work, but the strategy itself is agent reasoning.

## Key Concepts

- **Content Pillars** — 3-5 core topics that define the blog's focus
- **Content Calendar** — Scheduled publishing cadence with assigned topics
- **Topic Clusters** — Groups of related posts around a pillar topic for SEO
- **Content Audit** — Review of existing content for gaps, staleness, and performance
- **Editorial Cadence** — How often and when to publish (e.g., 3x/week, MWF at 9am UTC)
