# Content Distribution

Promote published content across social media, email newsletters, and other channels. This domain is workflow-driven — it reads content from emdash and uses external tools (Hermes, social APIs) for distribution.

## Reading Order

| Task | Start With | Then Read |
|------|-----------|-----------|
| Create social media posts | patterns.md | gotchas.md |
| Draft newsletter content | patterns.md | -- |
| Cross-promote content | patterns.md | -- |

## Key Concepts

- **Hermes Integration** — Paperclip's messaging bridge for Telegram, Discord, Slack, WhatsApp, etc.
- **Content Repurposing** — Transform a blog post into multiple format-specific pieces
- **Distribution Channels** — Social media, email, messaging platforms, syndication
- **Post URL** — Construct from `$EMDASH_URL` + collection path + slug

## Available Tools

| Channel | Tool | Notes |
|---------|------|-------|
| Telegram, Discord, Slack, etc. | Hermes (`mcp__hermes__messages_send`) | Via Paperclip Hermes bridge |
| Email newsletters | External tool (Mailchimp, ConvertKit, etc.) | Prepare content, agent can draft |
| RSS | Auto-generated by emdash | No action needed |
| Social media APIs | Platform-specific | Agent can generate copy |
