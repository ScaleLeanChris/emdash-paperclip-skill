# Content Distribution

Promote published content across social media, email newsletters, and other channels. This domain is workflow-driven — it reads content from emdash and uses external tools for distribution.

## Reading Order

| Task | Start With | Then Read |
|------|-----------|-----------|
| Create social media posts | patterns.md | gotchas.md |
| Draft newsletter content | patterns.md | -- |
| Cross-promote content | patterns.md | -- |

## Key Concepts

- **Content Repurposing** — Transform a blog post into multiple format-specific pieces
- **Distribution Channels** — Social media, email, messaging platforms, syndication
- **Post URL** — Construct from `$EMDASH_URL` + collection path + slug
- **Channel Tools** — Use whatever social/messaging tools are available in the agent's environment

## Available Tools

| Channel | How | Notes |
|---------|-----|-------|
| Social media | Platform APIs or social tools | Agent uses available social posting tools |
| Email newsletters | External tool (Mailchimp, ConvertKit, etc.) | Agent can draft content |
| RSS | Auto-generated by emdash | No action needed |
| Messaging platforms | Available messaging tools | Configured per-instance |
